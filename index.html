Truth Table Generator

Truth Table Notation Guide:

  • a or b - Logical OR
  • a and b - Logical AND
  • a xor b - Logical XOR
  • a = b - Logical IFF (if and only if)
  • a -> b - Logical Implication
  • (~a) - Negation
  • true - True constant
  • false - False constant
  • Use parentheses for grouping, e.g., (a or b) and c
  • Separate multiple statements using commas, e.g., (a or b), (b and c)
  • If the file doesn't download, put the statements in parenthesis, e.g., (a = b) = c instead of a = b = c

Note: Ensure spaces between variables and logical operands (e.g., "a and b").
